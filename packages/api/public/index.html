SDK available
import { EngramClient } from 'engram-sdk'

const client = new EngramClient({
  privateKey: process.env.AGENT_KEY,
  network: 'base-sepolia',
  apiUrl: 'https://engram.dkta.dev',
})

// register once, store forever
const { agentId } = await client.register()

const { hash } = await client.store({
  type: 'kv',
  data: { preference: 'dark mode' },
})

const { data } = await client.retrieve(hash)
Why Engram
🔐

Wallet-as-identity

Your wallet is your identity. AES-256-GCM keys derived from your signature. No key escrow, no third-party access.

⬡

Content-addressed storage

Every memory is encrypted and stored by its SHA-256 hash. Integrity-verified on every read.

⚡

x402 micropayments

Pay per operation in USDC on Base. No subscriptions, no accounts. Fractions of a cent per call.

📋

On-chain registry

Agent registration on Base. Your identity and memory index live on-chain, accessible from anywhere.

🤖

Agent-native

Designed for autonomous operation. No human-in-the-loop, no checkout flows, no browser required.

🌐

Any agent, anywhere

Any EVM wallet. Any HTTP client. TypeScript SDK included, or call the API from any language.
