# Engram v2 Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** A public shared memory API for agents — write memories tagged by project and agent, search them with FTS. All operations are x402-gated; the payer's wallet address is the sole identity. No API keys, no auth layer, no signup.

**Architecture:** Stateless Fastify API over PostgreSQL with FTS via tsvector. x402 payment middleware gates every route — `req.payment.from` (payer wallet address) is the owner identity. Private memories are scoped to that address. Rate limited per IP. Runs on VPS behind Caddy at engram.dkta.dev.

**Tech Stack:** Node.js 20, Fastify 4, PostgreSQL 16, Docker Compose, Caddy (existing), x402-express

**Repo:** `/home/hermes/engram` — gut existing packages, rebuild from scratch under `packages/api`

## Schema

```sql
CREATE TABLE memories (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  content       TEXT NOT NULL,
  tags          TEXT[] NOT NULL DEFAULT '{}',
  project       TEXT,
  agent_id      TEXT,
  visibility    TEXT NOT NULL DEFAULT 'private'
                CHECK (visibility IN ('public', 'private')),
  owner_address TEXT NOT NULL,              -- wallet address from x402 payment
  created_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
  tsv           TSVECTOR GENERATED ALWAYS AS (to_tsvector('english', content)) STORED
);

CREATE INDEX memories_tsv_idx       ON memories USING GIN(tsv);
CREATE INDEX memories_visibility_idx ON memories(visibility);
CREATE INDEX memories_owner_idx     ON memories(owner_address);
CREATE INDEX memories_project_idx   ON memories(project);
CREATE INDEX memories_created_idx   ON memories(created_at DESC);
```

## API Surface

```
POST   /memories              write a memory (x402 required, $0.001)
GET    /memories/search?q=... search memories (x402 required, $0.001)
GET    /memories/:id          get one memory (x402 required, $0.0001)
PATCH  /memories/:id          update own memory (x402 required, $0.001)
DELETE /memories/:id          delete own memory (x402 required, $0.0001)
GET    /health                healthcheck (free)
```

All routes except `/health` require a valid x402 payment. `req.payment.from` is the payer wallet address and is the sole identity for ownership checks.

Public memories are readable by any paying address. Private memories are only returned to the `owner_address` that wrote them.

### Task 3: x402 payment middleware

**Objective:** Gate all non-health routes with x402. Extract `req.payment.from` as the payer identity on each request.

**Files:**
- Create: `packages/api/src/plugins/payment.js`

**Step 1: Create `src/plugins/payment.js`**
```js
import fp from 'fastify-plugin'
import { paymentMiddleware } from 'x402-express'
import { createCdpAuthHeaders } from '@coinbase/x402'

const PAYMENT_ADDRESS = process.env.PAYMENT_ADDRESS
const NETWORK = process.env.NETWORK || 'base'

const facilitator = {
  url: 'https://api.cdp.coinbase.com/platform/v2/x402',
  createAuthHeaders: createCdpAuthHeaders(
    process.env.CDP_API_KEY_ID,
    process.env.CDP_API_KEY_SECRET
  )
}

// x402 route price map
export const PRICES = {
  'POST /memories':       { price: '$0.001',  network: NETWORK, config: { description: 'Write a memory' } },
  'GET /memories/search': { price: '$0.001',  network: NETWORK, config: { description: 'Search memories' } },
  'GET /memories/:id':    { price: '$0.0001', network: NETWORK, config: { description: 'Read a memory' } },
  'PATCH /memories/:id':  { price: '$0.001',  network: NETWORK, config: { description: 'Update a memory' } },
  'DELETE /memories/:id': { price: '$0.0001', network: NETWORK, config: { description: 'Delete a memory' } },
}

export default fp(async function paymentPlugin(fastify) {
  // Attach x402 middleware as a Fastify hook
  // paymentMiddleware is Express-style — wrap for Fastify compatibility
  const middleware = paymentMiddleware(PAYMENT_ADDRESS, PRICES, facilitator)

  fastify.addHook('onRequest', (req, reply, done) => {
    if (req.url === '/health') return done()
    middleware(req.raw, reply.raw, done)
  })

  // Decorate request with payer address extracted from x402 payment header
  fastify.decorateRequest('payerAddress', null)
  fastify.addHook('preHandler', (req, reply, done) => {
    // x402-express attaches payment info to req.raw after verification
    req.payerAddress = req.raw?.payment?.from ?? null
    done()
  })
})
```

### Task 4: Write and update memories

**Objective:** `POST /memories` stores content, `PATCH /memories/:id` updates own memory.

**Files:**
- Create: `packages/api/src/routes/memories.js`

```js
export default async function memoriesRoutes(fastify) {

  // POST /memories — write
  fastify.post('/memories', async (req, reply) => {
    const { content, tags = [], project, agent_id, visibility = 'private' } = req.body

    if (!content?.trim()) return reply.code(400).send({ error: 'content is required' })
    if (!['public', 'private'].includes(visibility)) {
      return reply.code(400).send({ error: 'visibility must be public or private' })
    }

    const [memory] = await fastify.sql`
      INSERT INTO memories (content, tags, project, agent_id, visibility, owner_address)
      VALUES (${content}, ${tags}, ${project ?? null}, ${agent_id ?? null}, ${visibility}, ${req.payerAddress})
      RETURNING id, content, tags, project, agent_id, visibility, owner_address, created_at
    `
    return reply.code(201).send(memory)
  })

  // PATCH /memories/:id — update own memory
  fastify.patch('/memories/:id', async (req, reply) => {
    const { content, tags, project, agent_id, visibility } = req.body ?? {}

    const [existing] = await fastify.sql`
      SELECT id, owner_address FROM memories WHERE id = ${req.params.id}
    `
    if (!existing) return reply.code(404).send({ error: 'Not found' })
    if (existing.owner_address !== req.payerAddress) {
      return reply.code(403).send({ error: 'Forbidden' })
    }

    const [updated] = await fastify.sql`
      UPDATE memories SET
        content    = COALESCE(${content ?? null}, content),
        tags       = COALESCE(${tags ?? null}, tags),
        project    = COALESCE(${project ?? null}, project),
        agent_id   = COALESCE(${agent_id ?? null}, agent_id),
        visibility = COALESCE(${visibility ?? null}, visibility),
        updated_at = now()
      WHERE id = ${req.params.id}
      RETURNING id, content, tags, project, agent_id, visibility, owner_address, created_at, updated_at
    `
    return updated
  })
}
```

### Task 5: Search, read, delete

**Objective:** `GET /memories/search`, `GET /memories/:id`, `DELETE /memories/:id`.

Add to `src/routes/memories.js`:

```js
  // GET /memories/search
  fastify.get('/memories/search', async (req, reply) => {
    const { q, project, agent_id, limit = 20, offset = 0 } = req.query
    if (!q?.trim()) return reply.code(400).send({ error: 'q is required' })

    const lim = Math.min(Number(limit), 100)
    const off = Number(offset)

    const results = await fastify.sql`
      SELECT id, content, tags, project, agent_id, visibility, owner_address, created_at,
             ts_rank(tsv, query) AS rank
      FROM memories, plainto_tsquery('english', ${q}) query
      WHERE tsv @@ query
        AND (
          visibility = 'public'
          OR (visibility = 'private' AND owner_address = ${req.payerAddress})
        )
        ${project   ? fastify.sql`AND project  = ${project}`   : fastify.sql``}
        ${agent_id  ? fastify.sql`AND agent_id = ${agent_id}`  : fastify.sql``}
      ORDER BY rank DESC, created_at DESC
      LIMIT ${lim} OFFSET ${off}
    `
    return { results, count: results.length, offset: off }
  })

  // GET /memories/:id
  fastify.get('/memories/:id', async (req, reply) => {
    const [memory] = await fastify.sql`
      SELECT id, content, tags, project, agent_id, visibility, owner_address, created_at, updated_at
      FROM memories WHERE id = ${req.params.id}
    `
    if (!memory) return reply.code(404).send({ error: 'Not found' })
    if (memory.visibility === 'private' && memory.owner_address !== req.payerAddress) {
      return reply.code(404).send({ error: 'Not found' })
    }
    return memory
  })

  // DELETE /memories/:id
  fastify.delete('/memories/:id', async (req, reply) => {
    const result = await fastify.sql`
      DELETE FROM memories
      WHERE id = ${req.params.id} AND owner_address = ${req.payerAddress}
    `
    if (result.count === 0) return reply.code(404).send({ error: 'Not found or not yours' })
    return reply.code(204).send()
  })
```

### Task 6: Wire everything in index.js

**Objective:** Register all plugins and routes, CORS, rate limiting, healthcheck, start server.

**`src/index.js`**
```js
import Fastify from 'fastify'
import rateLimit from '@fastify/rate-limit'
import cors from '@fastify/cors'

import paymentPlugin from './plugins/payment.js'
import memoriesRoutes from './routes/memories.js'
import sql from './db.js'

const fastify = Fastify({ logger: true })

fastify.decorate('sql', sql)

await fastify.register(cors, { origin: true })
await fastify.register(rateLimit, {
  max: Number(process.env.RATE_LIMIT_MAX ?? 200),
  timeWindow: Number(process.env.RATE_LIMIT_WINDOW ?? 60000),
  keyGenerator: (req) => req.ip,
})
await fastify.register(paymentPlugin)
await fastify.register(memoriesRoutes)

fastify.get('/health', async () => ({ status: 'ok', version: '2.0.0' }))

try {
  await fastify.listen({ port: Number(process.env.PORT ?? 3000), host: '0.0.0.0' })
} catch (err) {
  fastify.log.error(err)
  process.exit(1)
}
```
